----------------

python3.12 -m venv .venv

python -m pip install --prefer-binary -r backend/requirements.txt
cd backend

node --check wordpress-plugin/sustainable-catalyst-decision-studio/assets/js/scds-decision-studio.js

RENDER

  /health

WORDPRESS
---------
Upload sustainable-catalyst-decision-studio-plugin-v1.11.0.zip and replace the
existing plugin. Activation upgrades the Decision Studio database to 1.5.0 and
creates the room, member, and event tables.

Recommended private room shortcode:
  [sc_decision_studio mode="room" title="Collaborative Decision Room"]

Admin checks:
  SC Decision Studio > Collaborative Decision Rooms
  SC Decision Studio > Release Diagnostics

Confirm the plugin and backend both report 1.11.0 before enabling backend-routed
collaboration actions.
