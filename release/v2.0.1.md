# Decision Studio v2.0.1 — Catalyst Module Navigation and Handoff Repair

This patch restores visible access to the seven specialized Catalyst modules inside Decision Studio while preserving every v2.0.0 connected-platform workflow.

## Included

- First-class cards for Catalyst Canvas, Catalyst Data, Catalyst Analytics R, Global Impact Catalyst, Narrative Risk, Catalyst Finance, and Catalyst Grit.
- Open Module, Send Decision Packet, and Import Artifact actions.
- Configurable WordPress routes for all seven modules.
- Browser-local `scds-catalyst-module-handoff/1.0` records with explicit return URLs and no silent external transmission.
- WordPress and FastAPI module-navigation endpoints.
- Responsive styles, status feedback, contracts, samples, release manifests, documentation, and validation.
- Updated WordPress page HTML explaining the connected module workflow.
