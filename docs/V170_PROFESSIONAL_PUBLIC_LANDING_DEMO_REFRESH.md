# Decision Studio v1.7.0 — Professional Public Landing Page and Demo Refresh

Decision Studio v1.7.0 turns the public presentation layer into a launch-ready platform page and demo system. The release adds landing and demo shortcode modes, public template endpoints, admin guidance, and documentation that explains Decision Studio as the orchestration layer for Sustainable Catalyst modules.

## Added

- Public landing shortcode: `[sc_decision_studio mode="landing"]`
- Public demo shortcode: `[sc_decision_studio mode="demo"]`
- Public template endpoints for landing and demo pages
- Admin Public Landing & Demo page
- Professional workflow copy: Canvas → Data → Analytics R → Global Impact → Narrative Risk → Finance → Grit → Decision Studio
- Launch-ready positioning for Decision Packet, readiness, audit, scenario comparison, Workbench handoff, saved packets, and export center

## Use

Use the landing shortcode on `/platform/decision-studio/` and the demo shortcode on a dedicated demo page or platform demos page.
