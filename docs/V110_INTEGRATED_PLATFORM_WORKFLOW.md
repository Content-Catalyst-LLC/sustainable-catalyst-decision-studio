# Decision Studio v1.1.0 — Integrated Platform Workflow

Decision Studio v1.1.0 turns the Studio into the synthesis layer for the Sustainable Catalyst platform workflow.

## Workflow

Canvas → Data → Analytics R → Global Impact → Narrative Risk → Finance → Grit → Decision Studio

## Included in v1.1.0

- Integrated Workflow tab in the public shortcode UI.
- Module cards for Catalyst Canvas, Catalyst Data, Catalyst Analytics R, Global Impact Catalyst, Narrative Risk, Catalyst Finance, Catalyst Grit, and Decision Studio.
- Decision Packet template shared by frontend, WordPress REST, and backend API.
- Backend endpoints for module map and packet template.
- WordPress REST routes for module map and packet template.
- Admin Integrated Workflow page.

## Boundary

This release creates the workflow scaffold and Decision Packet contract. Full one-click artifact import from each module is planned for later builds.
