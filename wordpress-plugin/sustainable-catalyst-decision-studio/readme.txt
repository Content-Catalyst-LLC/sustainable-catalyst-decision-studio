Accessible and offline-capable Decision Studio with recovery snapshots, migration assessment, production release gates, public APIs, institutional integration, monitoring, publication, governance, and collaboration.

== Description ==

Decision Studio v1.16.0 adds accessibility validation, IndexedDB draft autosave, localStorage fallback, online/offline status announcements, secret-stripped recovery snapshots, additive packet migration assessment, and explicit performance, backup, restore, privacy, and human release gates.

Automated checks do not establish WCAG conformance or certification. Human assistive-technology testing and release authorization remain required.
